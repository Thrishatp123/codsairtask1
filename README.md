# Django CRUD Website

This project is a Django-based website that implements CRUD (Create, Read, Update, Delete) operations. It provides a simple interface for managing data stored in a database. create views and templates to almost match the provided design.

## Installation

1. Clone the repository:

    ```
    https://github.com/Thrishatp123/codsairtask1/tree/main/CRUD/mypro
    ```

2. Navigate into the project directory:

    ```
    cd mypro
    ```

3. Install the required dependencies:

    ```
    pip install -r requirements.txt
    ```

4. Apply database migrations:

    ```
    python manage.py migrate
    ```

## Usage

1. Start the Django development server:

    ```
    python manage.py runserver
    ```

2. Open a web browser and navigate to `http://127.0.0.1:8000/` to access the website.

3. Use the website interface to perform CRUD operations on the  table data .

## Features

- **Create:** Add new user to the database.
- **Read:** View existing entries stored in the database.
- **Update:** Modify existing entries in the database.
- **Delete:** Remove user from the database.
